HOW TO RUN THIS SCRIPT:
*** Prepare Hadoop, files and data:
	Connect to the ECS system
	Transfer requirements.txt, env_vars.sh, code.py and the data.csv files into the local file system
	Active the relevant environment variables by running env_vars.sh via:
		source env_vars.sh
*** Create a new venv
*** Install requirements via:
	pip install -r requirements.txt
*** Run script from python as below:
	* python code.py
